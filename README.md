# KBC Game Sound Project (Python)

## Description
This is a simple beginner-level Python project that plays a sound file using the `playsound` library.
It simulates a Kaun Banega Crorepati (KBC) style clock sound.

## Files
- kbc_game.py : Python script
- Kbc_Clock.mp3 : Sound file
- README.md : Project documentation
- requirements.txt : Required library

## How to Run
1. Install dependency:
   pip install playsound
2. Run the script:
   python kbc_game.py
